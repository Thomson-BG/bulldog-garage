PUT THE SITE ONLINE

HOST IT ONCE

For a classroom of Chromebooks, the site needs to live at a web address. Hosting it takes a couple of minutes, costs nothing, and after that every student just opens a link — quizzes, videos, and everything else work with zero setup on their end.

Why hosting? Chromebooks can't run a local server, and you can't put a multi-gigabyte folder on each one. A hosted link is the only practical classroom setup — and it's genuinely the easiest path.
EASIEST

Option 1 — Netlify Drop (drag & drop, ~2 min)
01
Go to app.netlify.com/drop in your browser
No account needed to start. (Make a free account to keep the link permanent and get a custom name.)
02
Drag the entire bulldog-garage folder onto the page
It uploads the whole site — pages, PDFs, videos, quizzes. Larger uploads take a few minutes the first time.
03
Netlify gives you a web address
Something like bulldog-garage.netlify.app. Share that link with students — done. Everything works.
—
Updating later
Drag the folder again to publish changes, or connect a free account for one-click redeploys.
DISTRICT

Option 2 — Your district / school web server
01
Ask your site tech or district IT to host the folder
It's a plain static site — no database, no server software, no PHP. Any web host or district web space serves it as-is.
02
They drop the bulldog-garage folder into your web space
This keeps everything on district infrastructure and under hemetusd.org — ideal for filtering and single sign-on.
GOOGLE

Option 3 — Other free static hosts
—
Vercel — vercel.com
Drag-and-drop or connect a folder; free, fast, reliable.
—
Cloudflare Pages — pages.cloudflare.com
Generous free tier, great for large media.
—
GitHub Pages — pages.github.com
Free; best if you're comfortable with a Git repo. Note: very large video libraries may exceed its size limits.
Videos & hosting size: the full site with all training videos is large (several GB). Netlify, Vercel, and Cloudflare Pages all handle it, but the first upload takes a few minutes. If you want a lighter deploy, you can host the site without the videos/ folder and keep videos on Google Drive or YouTube — ask and this can be set up.
