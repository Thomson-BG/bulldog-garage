This quiz needs the site to be online

You're opening the site as a local file. Interactive quizzes (and the safety tests) only run when the site is served from a web address — that's a browser security rule, and it's the same reason this needs to be hosted for your Chromebooks.

Recommended · one-time setup

Put it online — free

Host the site once and every student just opens a link. No installs, no downloads, works on any Chromebook. Quizzes and videos all work automatically.

  1. Go to app.netlify.com/drop
  2. Drag the whole bulldog-garage folder onto the page
  3. Netlify gives you a web address — share it with students
